🚀 QUICK START (2 minutes)

Step 1: Start Backend
─────────────────────────────────────────────────────────────────────────────
  cd backend
  npm run dev
  
  Expected:
    ✅ MongoDB connected
    🚀 Server running on http://localhost:37218

Step 2: Load Extension in Chrome
─────────────────────────────────────────────────────────────────────────────
  1. Open Chrome → chrome://extensions/
  2. Toggle "Developer mode" (top right)
  3. Click "Load unpacked"
  4. Select: chrome-extension folder
  5. Click "Select Folder"
  
  Result: ContextMind icon appears in toolbar

Step 3: Test It
─────────────────────────────────────────────────────────────────────────────
  1. Open multiple tabs:
     - https://claude.ai (AI conversation)
     - https://github.com (work)
     - https://youtube.com (noise)
  
  2. Click ContextMind icon in toolbar
  
  3. In popup, verify:
     ✅ Claude tab is checked (blue "AI" badge)
     ✅ GitHub tab is checked
     ❌ YouTube tab is unchecked (gray "Noise" badge)
  
  4. Click "Save Context" button
  
  5. Watch status messages:
     ⏳ Scanning selected tabs...
     ⏳ Extracting conversations...
     ⏳ Fetching structured data...
     ⏳ Compressing context...
     ⏳ Saving to database...
  
  6. Success! See:
     ✅ Green success message
     📋 Summary card: "You were working on..."
     📋 Copy button: "Copy Prompt (Paste into new AI)"

Done! Your context is saved in MongoDB. 🎉

⚙️ ARCHITECTURE

User Browser
    ↓
Chrome Extension Popup
    ├─ Lists all open tabs
    ├─ Shows classification badges (AI/work/noise)
    └─ User selects which tabs to save
    ↓
Content Scripts (extract tab text)
    ↓
POST /save API (backend)
    ├─ Process tabs (compress, extract, fallback)
    ├─ Generate summary
    ├─ Generate priming prompt
    └─ Store in MongoDB
    ↓
Response with session ID
    ↓
Extension shows summary + copy button
    ↓
User copies priming prompt
    ↓
Pastes into new AI chat → No re-explaining needed! ✨

🔑 KEY FEATURES

✅ One-Click Context Saving
   Click button → Everything captured

✅ Smart Tab Classification
   • AI domains: claude.ai, openai.com, gemini.google.com, etc
   • Work domains: github.com, notion.com, etc
   • Noise domains: youtube.com, netflix.com, twitter.com, etc

✅ Intelligent Content Extraction
   • AI conversations: extracted & compressed
   • Other content: MCP extraction or raw text fallback

✅ Priming Prompts
   Copy → Paste into new AI → Ready to resume

✅ MongoDB Persistence
   Sessions stored forever, retrieve anytime

✅ Error Handling
   Graceful fallbacks, never lose data

🐛 TROUBLESHOOTING

Backend won't start?
  Error: MONGO_URI not provided
  → Check backend/.env has MongoDB connection string

Extension not loading?
  → Reload: chrome://extensions/ → click ↻ next to ContextMind

No tabs showing in popup?
  → Reload extension: chrome://extensions/ → click ↻
  → Open new tab and try again

Content not saving?
  → Open DevTools (F12) → Console
  → Look for red error messages
  → Check backend console for details

Port 37218 in use?
  lsof -ti:37218 | xargs kill
  npm run dev
